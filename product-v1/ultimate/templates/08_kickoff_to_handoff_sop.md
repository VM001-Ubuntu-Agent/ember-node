# SOP: Project Kickoff to Handoff (AI Freelancer)

1. Run discovery call script.
2. Send scoped proposal with boundary clauses.
3. Collect payment + signed acceptance.
4. Run onboarding questionnaire.
5. Verify access checklist complete.
6. Build milestone 1 and send update.
7. Use change order for out-of-scope asks.
8. Deliver final handoff package.
9. Offer support retainer upsell.
