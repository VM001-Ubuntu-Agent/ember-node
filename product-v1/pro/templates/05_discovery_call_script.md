# Discovery Call Script (High-Intent)

Goal: qualify fast, avoid bad-fit clients.

1) "What business result do you need most in next 30-60 days?"
2) "What happens if this is not solved?"
3) "Who approves budget and timeline?"
4) "What tools and data access are ready right now?"
5) "What does success look like in numbers?"

Close line:
"Based on this, I can send a scoped proposal with exact deliverables and boundaries by [time]."
