# Metric Rules - First 7 Days

Goal: find proof fast. Not perfect branding. Proof.

## Daily numbers to track

Track these once per day:
- Posts/DMs sent
- Product page visitors
- Email opt-ins
- Sales by tier
- Revenue

## Minimum daily activity

Do this every day for 7 days:
- 3 public posts
- 10 helpful replies/comments
- 5 targeted DMs only if natural and not spammy
- 1 offer tweak if visitors are coming but no sales

## Decision rules

### If page visitors are under 30/day
Problem = traffic.
Fix = more posts, more comments, better platforms.

### If visitors are 30+/day but no sales
Problem = offer/page.
Fix = stronger headline, clearer pain, better examples, lower friction.

### If people ask questions but do not buy
Problem = trust or clarity.
Fix = add FAQ, examples, screenshots, stronger guarantee if platform supports it.

### If Basic sells but Pro does not
Problem = Pro value unclear.
Fix = make Pro the recommended tier and show why change-order template saves money.

### If Pro sells
Good. Double down. Make more content around scope creep.

## 150 visitor rule

Do not judge too early.
Wait until 150 targeted visitors before major pivot.
Then change ONE thing:
1. Headline
2. Niche angle
3. Tier pricing
4. Bonus stack

One change at a time.
